Estrutura for (para)

Esse tipo de laço é utilizado quando você sabe exatamente quantas vezes deseja que o código seja repetido. Embora ele seja mais comum para repetições finitas, também pode se tornar um laço infinito, caso a condição de continuidade não seja alterada corretamente.


Uma declaração for é feita da seguinte maneira:

1-)A expressão inicial é configurada e, se possível, executada. Geralmente, essa expressão serve para inicializar contadores, mas pode incluir qualquer tipo de expressão, inclusive declarações de variáveis.


2-)A expressão de condição é então verificada. Se a condição for verdadeira, o laço continua a ser executado. Caso contrário, o laço é encerrado. Se a condição for omitida, ela será considerada verdadeira por padrão.


3-)A instrução do laço é então executada. Se for necessário executar várias instruções, utilize um bloco de código ({ ... }) para agrupá-las.


4-)Por fim, a expressão de atualização (se houver) é aplicada, e o controle retorna para a verificação da condição para repetir o processo.
