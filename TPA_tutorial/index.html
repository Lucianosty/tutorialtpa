O que seria uma estrutura de repetição no JavaScript?

Estruturas de repetição no JavaScript são essenciais em praticamente todo algoritmo, pois permitem repetir uma ou mais instruções múltiplas vezes até que uma condição seja atendida. São muito importantes para otimizar a programação, especialmente quando você precisa executar uma tarefa várias vezes.

Saída:

Neste caso, simulamos a tabuada do 5, a variável i é o nosso contador, iniciamos ela com o valor 0 , portanto a condição será verdadeira até que ela atinja o valor 11, e por isso ela é iterada a cada repetição - i++, ao atingir tal valor o laço é quebrado.
