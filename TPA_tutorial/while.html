Estrutura while

Esse tipo de laço é normalmente utilizado quando não se sabe antecipadamente o número exato de repetições necessárias. O laço while é frequentemente empregado em situações como a leitura de entradas do usuário, onde o código continua sendo executado enquanto uma condição específica não for atendida, como quando o usuário digita um valor determinado (como "x") para encerrar o programa.


Uma declaração while é feita da seguinte maneira:

Quando a condição se torna falsa, o laço é interrompido e o controle é transferido para a instrução que vem depois dele.


O teste da condição ocorre antes de cada execução do laço. Se a condição for verdadeira, o laço é executado e a condição é verificada novamente. Se a condição for falsa, o laço termina e o controle passa para as instruções que estão após o laço.


Caso precise executar várias instruções dentro do laço, você pode usar um bloco de código ({ ... }) para agrupar essas instruções.
